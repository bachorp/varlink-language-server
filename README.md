# Varlink Language Server

Features:

- Diagnostics (including workspace)
    - Syntax errors
    - Missing/duplicate declarations
- Symbols (including workspace)
- Semantic tokens
- Go to definition
- Formatting via [`varlinkfmt`](https://crates.io/crates/varlinkfmt)
- Completion
- Hover
- Selection range
- Rename symbol
- References
